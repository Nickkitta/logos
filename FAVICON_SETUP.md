# Настройка иконки сайта (Favicon)

## Текущая иконка

Создана простая SVG иконка с книгой - символ образования.

## Как заменить на свою

### Вариант 1: Использовать SVG (рекомендуется)

1. Создай SVG файл с иконкой (100x100px)
2. Замени файл `public/favicon.svg`
3. Готово! SVG масштабируется идеально на всех размерах

### Вариант 2: Использовать PNG

1. Создай PNG иконки разных размеров:
   - `favicon-32x32.png` (32x32px)
   - `favicon-16x16.png` (16x16px)
   - `apple-touch-icon.png` (180x180px) - для iOS
   
2. Положи их в папку `public/`

3. Обнови `nuxt.config.ts`:
```typescript
link: [
  { rel: 'icon', type: 'image/png', sizes: '32x32', href: '/favicon-32x32.png' },
  { rel: 'icon', type: 'image/png', sizes: '16x16', href: '/favicon-16x16.png' },
  { rel: 'apple-touch-icon', sizes: '180x180', href: '/apple-touch-icon.png' }
]
```

### Вариант 3: Использовать генератор

Используй онлайн генераторы:
- https://realfavicongenerator.net/ - создаст все нужные размеры
- https://favicon.io/ - простой генератор из текста или эмодзи

## Размеры иконок

- **16x16** - старые браузеры
- **32x32** - стандартный размер для вкладок
- **180x180** - iOS (apple-touch-icon)
- **192x192** - Android
- **512x512** - PWA

## Цвета

Текущая иконка использует:
- Фон: `#00DC82` (зеленый, основной цвет сайта)
- Книга: белый
- Закладка: `#FFD700` (золотой)

Можешь изменить цвета в `public/favicon.svg`

## Проверка

После изменения:
1. Очисти кэш браузера (Ctrl+Shift+R)
2. Проверь вкладку браузера
3. Проверь закладки
4. Проверь на мобильном (добавь на главный экран)
