# Пошаговая настройка Telegram для контактной формы

## Шаг 1: Создать бота

1. Открой Telegram
2. Найди **@BotFather** (официальный бот для создания ботов)
3. Отправь команду: `/newbot`
4. Введи название бота (например: "School Contact Bot")
5. Введи username бота (должен заканчиваться на `bot`, например: `school_contact_bot`)
6. **Скопируй токен** который пришлет BotFather (выглядит так: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 2: Получить Chat ID

### Способ 1 (самый простой):
1. Найди в Telegram бота **@userinfobot**
2. Нажми Start или отправь любое сообщение
3. Бот пришлет твой Chat ID (число, например: `123456789`)

### Способ 2 (через своего бота):
1. Найди своего созданного бота в Telegram (по username)
2. Нажми "Start" или отправь `/start`
3. Открой в браузере: 
   ```
   https://api.telegram.org/bot<ТВОЙ_ТОКЕН>/getUpdates
   ```
   (замени `<ТВОЙ_ТОКЕН>` на токен из Шага 1)
4. Найди в JSON ответе: `"chat":{"id":123456789}`
5. Это число и есть твой Chat ID

## Шаг 3: Добавить в проект

1. Открой файл `.env` в корне проекта (если его нет - создай)
2. Добавь туда:
   ```env
   TELEGRAM_BOT_TOKEN=твой-токен-из-шага-1
   TELEGRAM_CHAT_ID=твой-chat-id-из-шага-2
   SCHOOL_EMAIL=info@school.ru
   ```

3. Сохрани файл

## Шаг 4: Проверить

1. Перезапусти dev сервер:
   ```bash
   npm run dev
   ```

2. Открой страницу контактов на сайте
3. Заполни и отправь форму
4. Проверь Telegram - должно прийти сообщение от твоего бота!

## Важно!

- ❌ **НЕ коммить** файл `.env` в Git
- ✅ Файл `.env` уже добавлен в `.gitignore`
- ✅ Можешь создать `.env.example` с примером (без реальных токенов)
- ✅ Токен и Chat ID храни только локально

## Если что-то не работает

1. Проверь, что токен и Chat ID правильно скопированы (без пробелов)
2. Убедись, что отправил хотя бы одно сообщение боту (нажал Start)
3. Проверь консоль браузера и терминал на ошибки
4. Попробуй отправить тестовое сообщение через браузер:
   ```
   https://api.telegram.org/bot<ТОКЕН>/sendMessage?chat_id=<CHAT_ID>&text=test
   ```

## Для продакшена

На сервере (например, Vercel, Netlify) добавь переменные окружения через их панель управления:
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`
- `SCHOOL_EMAIL`
