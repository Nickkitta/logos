# Настройка карты

## Как использовать

Карта уже работает! Просто укажи свои координаты:

```vue
<YandexMap 
  address="г. Москва, ул. Школьная, д. 15"
  :coordinates="[55.751244, 37.618423]"
  :zoom="16"
/>
```

## Как найти координаты своего адреса

### Способ 1: Яндекс.Карты
1. Открой https://yandex.ru/maps
2. Найди свой адрес
3. Кликни правой кнопкой на нужное место
4. Выбери "Что здесь?"
5. Скопируй координаты (формат: широта, долгота)

### Способ 2: Google Maps
1. Открой https://google.com/maps
2. Найди свой адрес
3. Кликни правой кнопкой на нужное место
4. Координаты появятся вверху (формат: широта, долгота)

### Способ 3: 2GIS
1. Открой https://2gis.ru
2. Найди свой адрес
3. Кликни на место
4. Координаты будут в URL

## Параметры компонента

```vue
<YandexMap 
  address="Адрес для подсказки"
  :coordinates="[широта, долгота]"
  :zoom="16"
/>
```

- `address` - текст, который показывается при клике на метку
- `coordinates` - массив [широта, долгота]
- `zoom` - уровень приближения (1-20, обычно 15-17 для зданий)

## Примеры координат

```javascript
// Москва, Красная площадь
[55.753544, 37.621211]

// Санкт-Петербург, Дворцовая площадь
[59.939095, 30.315868]

// Казань, Кремль
[55.798551, 49.106324]
```

## Функции карты

- Клик на карту → открывается в полноэкранном режиме
- В полноэкранном режиме можно:
  - Перемещаться
  - Приближать/отдалять
  - Кликнуть на метку для просмотра адреса
- Кнопка закрытия или клик вне карты → закрывает полноэкранный режим

## API ключ (необязательно)

Яндекс.Карты работают без ключа, но с ограничениями. Для продакшена лучше получить бесплатный ключ:

1. Зайди на https://developer.tech.yandex.ru/
2. Зарегистрируйся
3. Создай JavaScript API ключ
4. Добавь в компонент:

```javascript
script.src = 'https://api-maps.yandex.ru/2.1/?apikey=ВАШ_КЛЮЧ&lang=ru_RU'
```

## Альтернативы

Если нужна другая карта:
- **Google Maps** - платная, но популярная
- **OpenStreetMap** - бесплатная, открытая
- **2GIS** - хорошо для России и СНГ
