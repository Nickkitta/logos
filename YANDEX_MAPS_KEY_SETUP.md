# Настройка API ключа Яндекс.Карт

## Проблема: Invalid API key

Если видишь ошибку "Invalid API key", проверь:

### 1. Правильный тип ключа

На https://developer.tech.yandex.ru/ нужно создать именно:
- ✅ **JavaScript API и HTTP Геокодер**
- ❌ НЕ "Геокодер" 
- ❌ НЕ "Static API"

### 2. Добавь домены

В настройках ключа добавь разрешенные домены:
```
http://localhost:3000
http://localhost:*
```

Для продакшена добавь свой домен:
```
https://твой-сайт.ru
```

### 3. Подожди активации

После создания ключа может потребоваться 5-10 минут для активации.

### 4. Проверь формат ключа

Ключ должен быть в формате UUID:
```
12345678-1234-1234-1234-123456789abc
```

## Работа без ключа

Яндекс.Карты работают без ключа, но с ограничениями:
- Максимум 25,000 запросов в сутки
- Водяной знак "Для разработки"

Для небольшого школьного сайта этого достаточно!

## Текущая настройка

Сейчас карта загружается **БЕЗ ключа**. Это нормально для разработки и небольших сайтов.

Если хочешь использовать ключ:
1. Создай правильный тип ключа
2. Добавь домены
3. Подожди активации
4. Раскомментируй код в `components/YandexMap.vue` (там есть инструкция)

## Альтернативы

Если Яндекс.Карты не подходят, могу добавить:
- **OpenStreetMap** (Leaflet) - полностью бесплатно, без ключей
- **2GIS** - хорошо для России, бесплатно
- **Google Maps** - платно, но популярно

Скажи если нужна другая карта!
